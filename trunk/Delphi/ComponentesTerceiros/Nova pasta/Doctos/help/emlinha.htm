Em Linha

Verifica se o ECF está em ligado.

Exemplo

while not EmLinha( 1 ) do { Impressora está em-linha ? }
..begin
.....if Retentar and (MessageDlg('A impressora '+ModeloStr+' não está pronta.'+#10+#10+ 'Deseja tentar novamente ?',mtConfirmation, [mbYes,mbNo],0) = mrYes) then
......Continue ;

.....raise ShowMessage( 'Impressora '+ModeloStr+' não está em linha') ;
..end ;
