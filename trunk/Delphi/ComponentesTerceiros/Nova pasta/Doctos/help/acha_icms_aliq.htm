Acha ICMS Aliquota

Verifica se aliquota está cadastrada. Deve ser passado a % da aliquota e caso o retorno seja = nil então a alíquota não está cadastrada.

Exemplo

Aliquota := AchaICMSAliquota( ValAliquota ) ;
.....if Aliquota <> nil then
........AliquotaStr := Aliquota.Indice
.....else
........raise ShowMessage( 'Aliquota nao encontrada');
